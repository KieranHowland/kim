KIM API TEXTS / RESPONSE EXAMPLES (response_examples.txt)

----------------------------------------------------------------------------------------

SUCCESS:
 {
   "status":200,
   "data":{
     "uploads":500
   }
 }

----------------------------------------------------------------------------------------

INTERNAL SERVER ERROR:
 {
   "status":500,
   "code":"internalServerError",
   "reference":"ABCDEFGH12345678"
 }

`reference` IS THE ERROR REFERENCE WHICH IS USED TO DIAGNOSE THE ISSUE

----------------------------------------------------------------------------------------

USER ERROR:
 {
   "status":400,
   "code":"invalidFileExtension"
 }

`code` IS THE ERROR CODE WHICH SUMMARISES THE ERROR IN A FEW WORDS

(SEE `error_codes`)
